## 3. 单词拼拼消（Word Match）游戏
- **核心文件**: `word-match-game.html`
- **线上地址**: https://silver-boba-6dd160.netlify.app
- **技术栈**: 单文件 HTML/CSS/JS，无外部依赖

### 核心机制
- 6x6 消消乐棋盘，交换相邻字母形成3连消除
- 消除目标字母以拼出指定单词
- 26关难度递进：3字母 → 10+字母
- 词库：中国九年义务教育英语课程词汇（260词，全配例句）

### 游戏模式（4种）
1. **闯关模式**：26关线性推进，每关主题色切换
2. **限时挑战**：60秒连击拼词
3. **无尽模式**：无限步数，每3词提升难度（干扰字母扩展）
4. **复习模式**：从已学单词池中出题

### 单词学习闭环
- 单词详情弹窗（ℹ️按钮）+ 发音（Web Speech API）
- 收藏系统（localStorage 持久化）
- "我的单词本"独立弹窗
- 已学单词记录（用于复习模式）
- 通关后展示例句 + 分享图片生成（HTML5 Canvas）

### 视觉与动效
- 8套动态主题（forest/ocean/candy/space/sunset/ice/fire/rainbow）
- 消除粒子爆开、飘分动画（+XX）
- Combo 指示器
- 提示脉冲光圈（hintPulse）
- 移动端响应式适配

### 难度曲线（v2）
- **步数**: `单词长度×4 + 12 + floor(关卡/2)`（第1关~24步，第26关~70步）
- **目标字母权重**: 60% → 50% → 42% → 35%（随关卡递减）
- **干扰字母**: XYZ → +QJ → +VK → +WF → +HM（逐步增加）
- **死锁处理**: 消除连锁后若无可行移动，自动重排棋盘
- **无尽模式**: 单词池随难度逐步解锁（避免早期出现长词）

### 音频
- Web Audio API 合成音效（swap/match/collect/win/lose/invalid/tick）
- TTS 单词/例句朗读
- 静音切换按钮

### 存档
- localStorage 全局存档：`wordMatchGlobal`
- 保存：最高关卡、总得分、已学单词列表、收藏列表

### 近期优化（积分与学习闭环）
- **学习型道具商店**：新增首字母提示、释义扩展卡、例句拆解卡、错词复活卡、复习加倍卡。
- **单词掌握度系统**：每个单词按正确、失败、复习次数计算掌握度，从“初见”到“已巩固”，在目标区、单词详情、单词本、学习报告中展示。
- **PWA 离线安装**：新增 `manifest.json`、`service-worker.js`、`icon.svg`，支持添加到手机桌面和基础离线访问。
- **技术整理**：保留单文件游戏主体，同时把 PWA 必需资源拆为独立文件，后续可继续拆分 `style.css/game.js/words.json`。
